# Nimbus_Project_Shreta_Ojha_4

Overview  
Nimbus_Project_Shreta_Ojha_4 is a simple hospital / patient-management and billing application written in C. It allows basic operations like adding, displaying, searching, and updating patient records â€” as well as billing and file handling operations.  

This project is intended as a console-based CLI program for managing patients and billing in a small clinic/hospital setup.

Features

- Add new patient records (name, age, medical history, contact, etc.)  
- Display all patient records  
- Search for existing patients by ID / name / attribute  
- Update patient records (edit details)  
- Basic billing functionality  
- File-based storage (data persists between runs)
